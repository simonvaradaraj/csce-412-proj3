\doxysubsection{Detailed Description}
A class representing a network request with input and output IPs, a time duration, and a job type. 

A class representing an incoming request.

The request class encapsulates the details of a network request, including the input and output IP addresses, the time associated with the request, and the type of job it represents. It provides a constructor for initializing these attributes and a method for converting the request data into a human-\/readable string format.
