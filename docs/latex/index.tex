 \hypertarget{index_autotoc_md1}{}\doxysection{Overview}\label{index_autotoc_md1}
This project implements a load balancer in C++ designed to efficiently distribute incoming network requests across multiple servers. The load balancer dynamically scales the number of servers based on the current load, ensuring optimal performance and resource utilization.\hypertarget{index_autotoc_md2}{}\doxysection{Features}\label{index_autotoc_md2}

\begin{DoxyItemize}
\item {\bfseries{Dynamic Scaling\+:}} Adjusts the number of servers based on request volume.
\item {\bfseries{Thread Management\+:}} Each server is represented as a separate thread.
\item {\bfseries{Load Distribution Algorithms\+:}} Supports various algorithms for load distribution (e.\+g., round-\/robin, least connections).
\item {\bfseries{Real-\/time Monitoring\+:}} Monitors server health and performance metrics.
\item {\bfseries{User-\/friendly Interface\+:}} Provides a simple command-\/line interface for configuration.
\end{DoxyItemize}\hypertarget{index_autotoc_md3}{}\doxysection{Getting Started}\label{index_autotoc_md3}
